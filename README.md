# 实用工具导航网站

这是一个集合了多种实用工具的导航网站，旨在提升用户的工作效率。

## 当前工具

### 1. 行列转换工具
- **功能**：快速进行数据的行转列、列转行操作
- **特点**：
  - 支持自定义分隔符
  - 不丢失任何数据行
  - 简洁易用的界面
  - 支持复制和下载结果

## 使用方法

1. 打开 `index.html` 文件
2. 点击相应的工具卡片
3. 按照工具界面的提示进行操作

## 目录结构

    ├── styles.css      # 行列转换工具样式
